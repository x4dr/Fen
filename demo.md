# Markdown Demo

A reference for local markdown features.

## Transclusion

Include content from another page: \[!demo#transclusion\]

## Foldable Sections

Prefix a heading with `!` to make it collapsible: ## !Heading

## !Demo Foldable

This content starts collapsed. Click the heading to expand it.

## Section Tooltip

Hover over \[!t:demo#transclusion\] to preview that section.

## Infolet

Show an item tooltip: \[!q:itemname\]

## Clock

Inline progress clock: [clock|progress|3|8]

## Glitch Text

Cyberpunk glitch effect: g~text~g or g~text~replacement~g

## Strikethrough

This is ~~strikethrough~~ text.

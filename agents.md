# Agents.md

This file documents the findings and workflows established during the collaboration between the User and the AI Agent.

## Collaboration Overview
- **Objective:** Build a consistent yet intentionally incongruent world-building system for a P&P setting.
- **Workflow:** 
    - Separate "Hard Rules" (mechanics/stats) from "Lore" (unreliable, multi-perspective).
    - Maintain existing documents for historical reference while gradually refining the structure.
    - Focus on modular expansion of lore and rulesets.

## Established Structure
- `/lore/`: Unreliable narratives, cultural descriptions, and subjective world-building.
- `/rules/`: Objective mechanical systems, character options, and gameplay-defining stats.

## Findings
- Magic system is divided between "Scholastic Law" (Onyki's Laws) and "Practical Application" (Source/Concept/Aspects in `magic.md`).
- Racial information split into individual files in `/rules/races/` (mechanics) and `/lore/races/` (narrative), with cross-links.
